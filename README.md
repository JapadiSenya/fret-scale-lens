# fret-scale-lens

弦楽器(ベース/ギター)のフレットボード上に音名を表示し、指定したキー・スケールに対応するフレット位置をハイライト表示するツールです。ビルド不要の静的Webページとして、GitHub Pages上で動作する想定です。

## 主な機能

- **フレットボード表示**: SVGで指板を描画し、各フレットの音名をオクターブまで含めて計算・表示
- **可変チューニング**: 弦数・各弦の開放音を自由に設定可能。ベース4弦/5弦・ギター6弦の標準チューニングをプリセットとして用意し、カスタムチューニングも組める
- **ネックポジションマーク**: 3・5・7・9・12(2つ)・15・17・19・21・24(2つ)フレットにインレイマークを表示
- **キー・スケールハイライト**: ルート音とスケール(メジャー/ナチュラルマイナー/メジャーペンタトニック/マイナーペンタトニック/ブルース)を選択し、該当するフレット位置を色分け表示
- **機能和声表示**: スケール構成音表示と、度数(ローマ数字)・トニック/サブドミナント/ドミナントの機能和声表示をトグルで切り替え可能
- **音声再生**: 弦・フレットをタップ/クリックすると、Web Audio APIでその場で合成した音を再生(音声ファイル不使用)。全発音は共通のマスター音量スライダーで調整可能
- **TAB譜作成・再生**: 指板をクリックしてTAB譜を作成・編集・再生できる譜面エディタ機能
  - 音符長(全音符〜16分音符)・付点、休符、ゴーストノート(✕表示)の入力
  - タイ/ハンマリング・プリング/スライドのアーティキュレーション付与(再生にも反映)
  - 音符のクリック選択(Shiftで範囲選択)による削除・音符長/付点の一括変更・コピー&ペースト
  - Undo/Redo、選択位置からの再生、メトロノーム、1オクターブ上げ再生オプション
  - 小節線・小節番号の自動表示と、拍子に対して長さが合わない小節の警告表示(赤系背景)
  - TAB譜の数字にキー・スケール・表示モードに応じた配色を反映するオプション
  - TAB譜(単一曲)のJSON形式でのExport/Import(export時は画面上部の設定=チューニング・キー・スケール等のスナップショットも同梱し、import時は読み込めた項目のみ反映。読み込めなかった設定項目がある場合は画面上部にポップアップ通知。TAB譜本体は常にそのまま取り込む)、localStorageへの自動保存
  - 折りたたみ式のJSON直接編集エリア(貼り付け・編集内容を即座にTAB譜へ反映。再生中は編集不可。選択中/再生中の音符に対応する箇所をJSON側でもハイライト・自動スクロール。音符は1行1音符で整形表示し、小節の区切りには空行を挿入)
- **設定の永続化**: チューニング・フレット数・キー・スケール・表示モード・マスター音量・TAB再生オプション(オクターブ上げ・配色連動・メトロノームON/OFF)等をlocalStorageに保存し、次回訪問時に自動復元
- PC/タブレットでのマウス・タップ操作を想定(スマホ向けレスポンシブ対応は非対応)

## 使い方

下記「ローカルでの動作確認」の手順で静的サーバー経由で `index.html` を開くと利用できます。

操作の流れ:

1. 上部の「キー」「スケール」でハイライトしたいキー・スケールを選択
2. 「チューニングプリセット」で弦構成を選択、または「チューニング詳細設定」から弦の追加/削除・開放音・フレット数を変更
3. 「表示モード」でスケール構成音表示⇔機能和声(T/S/D)表示を切り替え
4. フレットボード上の丸をクリック/タップすると、その音が再生される
5. 音符長(必要なら付点も)を選んだ状態でフレットボードをクリックすると、下部のTAB譜に音符が追加される。休符・ゴーストノート・アーティキュレーション・削除・Undo/Redo・コピー&ペーストもTAB譜側のツールバーから操作できる
6. TAB譜の「再生」ボタンで作成した譜面を再生。音符を選択しておくとその位置から再生される

## 技術スタック

- Vanilla JavaScript(ES Modules)+ HTML + CSS
- SVGによるフレットボード描画
- Web Audio API(`OscillatorNode` + `GainNode`)による音声合成・TAB再生スケジューリング
- 外部フレームワーク・ビルドツールは不使用

## ファイル構成

```
/index.html
/style.css
/js/
  ├ notes.js         … 音名・周波数計算
  ├ scales.js        … スケール定義・ハイライト判定・機能和声判定
  ├ tuning.js        … 弦数・チューニング状態管理、プリセット定義
  ├ storage.js       … localStorage読み書き(設定・TAB譜ライブラリ)
  ├ audio.js         … Web Audio APIによる音再生・マスター音量
  ├ render.js        … フレットボードSVG描画
  ├ tab.js           … TAB譜データモデル・編集操作・Undo/Redo履歴
  ├ tabRender.js     … TAB譜表示エリアのDOM描画
  ├ tabPlayback.js   … TAB譜再生・メトロノームのスケジューリング
  └ main.js          … 初期化・イベントバインド
```

## ローカルでの動作確認

ES Modules(`<script type="module">`)を使用しているため、`index.html` を `file://` で直接開くとブラウザのCORS制約によりモジュール読み込みに失敗します。簡易的な静的サーバーを起動して確認してください。

```bash
# 例: Python
python -m http.server 8000

# 例: Node.js (npx)
npx serve .
```

起動後、`http://localhost:8000` にアクセスします。
